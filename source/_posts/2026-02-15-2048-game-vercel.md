> 第一次用 OhMyOpenCode 开发的游戏上线了！

## 🎮 游戏地址

**Playing now:** https://2048-game-sigma-wheat.vercel.app

## 🛠️ 技术栈

| 技术 | 说明 |
|------|------|
| **HTML5** | 语义化标签 |
| **CSS3** | 响应式布局、动画效果 |
| **JavaScript** | 原生 JS，无框架 |
| **Vercel** | 免费托管与部署 |

## 🎯 游戏特性

- ✅ 4x4 经典网格
- ✅ 键盘控制（方向键）
- ✅ 移动端触摸支持
- ✅ 分数与最高分记录（localStorage）
- ✅ 精美配色与动画
- ✅ 游戏结束检测

## 📱 响应式设计

完美支持手机端游玩！

## 🚀 部署过程

1. 创建项目目录
2. 编写 HTML + CSS + JS
3. GitHub 创建仓库
4. Vercel 一键部署

**Total time:** ~10 分钟
