## 恶性循环分析

### 八步走向深渊

```
1. 产品没竞争力 → 本品牌销量下跌
2. 销量跌 → 厂家还在生产 → 库存爆仓
3. 库存压死经销商 → 不敢拿车、退网
4. 渠道崩了 → 更卖不动 → 只能疯狂降价
5. 利润更少 → 没钱研发 → 产品更烂、服务更差
6. 产品更烂 → 更没竞争力 → 口碑销量继续滑坡
7. 厂家资金链紧张 → 拖供应商、砍员工 → 体系雪崩
8. 体系崩 → 品牌边缘化 → 退市
```
